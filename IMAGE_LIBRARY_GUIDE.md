# FDeck 图片库投喂指南

## 概述

FDeck采用**图片库优先 + AI生图兜底**的策略，大幅降低成本同时保证质量。

- **成本优化**：从~1元/PPT降至~0.1元/PPT（降低90%）
- **质量保证**：投喂模板经过筛选，质量有保证
- **风格一致**：同一模板的图片风格统一

## 投喂流程

### 方式一：自动提取（推荐）

1. **准备PPT文件**
   - 将优质PPT文件放入 `用户上传/` 目录
   - 文件命名规范：`主题_风格_序号.pptx`
     - 示例：`党建汇报_党政红_001.pptx`
     - 示例：`AI技术报告_科技蓝_001.pptx`

2. **运行提取脚本**
   ```bash
   # 安装依赖（首次）
   npm install
   
   # 批量提取用户上传目录下的所有PPT
   npm run extract
   
   # 或提取单个文件
   npm run extract-single 用户上传/xxx.pptx
   ```

3. **检查结果**
   ```bash
   # 查看图片库统计
   npm run stats
   
   # 输出示例：
   # 📊 图片库统计:
   #   总数: 150
   #   按风格: {"party_red":30,"tech_blue":40,"business":50,"warm":20,"nature":10}
   #   按类型: {"cover":30,"content":100,"ending":20}
   ```

### 方式二：手动添加

1. **准备图片**
   - 放入 `templates-assets/images/library/` 目录
   - 建议命名：`style_type_keywords.png`

2. **编辑索引**
   打开 `templates-assets/image-library-index.json`，添加：
   ```json
   {
     "id": "img_custom_001",
     "path": "templates-assets/images/library/custom_001.png",
     "type": "cover",
     "style": "tech_blue",
     "keywords": ["AI", "科技", "未来"],
     "colors": ["#1E90FF", "#00CED1"],
     "layout": "center",
     "mood": "科技感、未来感",
     "usage_count": 0,
     "quality_score": 4.5,
     "source": "manual"
   }
   ```

## 风格分类

### 1. 党政红金 (party_red)
- **主色调**：#C41E3A（中国红）、#FFD700（金色）
- **适用场景**：党建、党课、表彰、红色主题
- **情绪**：庄重大气、华贵典雅

### 2. 科技蓝 (tech_blue)
- **主色调**：#1E90FF、#00CED1、#191970
- **适用场景**：AI、互联网、数字化、创新
- **情绪**：科技感、未来感、专业严谨

### 3. 商务蓝白 (business)
- **主色调**：#2C5282、#4A90D9、#1A365D
- **适用场景**：企业汇报、方案、计划、总结
- **情绪**：简洁专业、清爽大气

### 4. 温暖米色 (warm)
- **主色调**：#D4A574、#F5DEB3、#8B7355
- **适用场景**：培训、教育、生活、健康
- **情绪**：温暖舒适、亲切自然

### 5. 清新绿色 (nature)
- **主色调**：#228B22、#90EE90、#006400
- **适用场景**：环保、自然、健康、有机
- **情绪**：自然清新、生机盎然

## 质量标准

### 图片质量要求
- ✅ 分辨率：≥1920x1080（推荐2560x1440）
- ✅ 格式：PNG（推荐）、JPG
- ✅ 色彩：清晰、鲜艳、无色差
- ✅ 内容：纯背景图，无文字、无图标

### 模板质量评分
- **5.0分**：顶级设计，可直接使用
- **4.5分**：优秀设计，微调后可用
- **4.0分**：良好设计，适合多数场景
- **3.5分**：合格设计，有改进空间
- **<3.5分**：不纳入图片库

## 匹配规则

### 匹配评分（满分100分）
- **风格匹配**：40分（完全匹配）/ 20分（兼容风格）
- **关键词匹配**：30分（根据主题关键词）
- **情绪匹配**：20分（氛围一致性）
- **质量+频率**：10分（质量分+使用频率）

### 使用策略
- **≥65分**：使用库中图片（免费）
- **<65分**：调用AI生图（~0.1元/张）

## 成本估算

### 图片库充足时
- 库图命中率：80%
- AI生图比例：20%
- **单PPT成本**：10页 × 20% × 0.1元 = **0.2元**

### 图片库不足时
- 库图命中率：30%
- AI生图比例：70%
- **单PPT成本**：10页 × 70% × 0.1元 = **0.7元**

### 纯AI生图
- **单PPT成本**：10页 × 0.1元 = **1.0元**

## 投喂建议

### 目标库容
- **初期**：每风格50张（共250张）
- **中期**：每风格100张（共500张）
- **长期**：每风格200张（共1000张）

### 投喂优先级
1. **高频场景**：商务、科技（优先投喂）
2. **热门主题**：AI、数字化、党建
3. **稀缺风格**：自然、温暖

### 投喂来源
- 优质PPT模板网站
- 设计师投稿
- 用户贡献（后续开发贡献功能）
