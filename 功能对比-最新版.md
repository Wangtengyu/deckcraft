## 二、核心差距详解

### 🔴 最高优先级：大纲生成流程

**原Skill流程**：
```
用户输入主题 → AI生成详细大纲 → 用户确认/编辑 → 生成PPT
```

**原Skill大纲内容**：
```json
{
  "title": "微服务迁移：小步快跑，持续验证",
  "pages": [
    {
      "type": "cover",
      "title": "微服务迁移实战指南"
    },
    {
      "type": "content",
      "section": "评估现状：摸清家底才能精准施策",
      "points": [
        "全面梳理现有系统边界和依赖关系",
        "识别核心业务模块与数据流向",
        "评估团队技术栈和运维能力"
      ]
    }
  ]
}
```

**DeckCraft当前**：
- 使用预设的 NARRATIVE_STRUCTURES 模板
- 没有AI生成大纲
- 标题和内容都是模板化的

**需要实现**：
1. 新增 `generate-outline` API（Laf上已有这个函数名，需确认功能）
2. 调用AI生成包含以下内容的大纲：
   - 每页标题（带观点）
   - 每页内容要点（具体说明+数据支撑）
   - 叙事结构适配
