# AI内容幻觉检测器

一个用于检测AI生成内容中幻觉、编造和数据失实问题的技能工具。

## 功能特点

- **识别高风险内容**：自动标注数据、引用、范围类高风险内容
- **系统化核验方法**：出处追溯、期待值校准、来源分级、逐项排查
- **结构化报告输出**：生成清晰的核验报告和验证清单

## 使用方式

触发词：`/verify` / `/核验` / `/检测幻觉` / `/真实性检查`

## 核验维度

1. 数据类：金额、增长率、百分比、数量统计
2. 引用类：政策文件、法规条文、报告引用
3. 范围类：全国、全球、全行业数据
4. 来源类：官方报道、主流媒体、非官方来源
