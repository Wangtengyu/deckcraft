# AI检测器技能包 🦞

> 帮助你识别AI生成内容中的幻觉、虚假信息和消费陷阱

## 📦 包含技能

### 1️⃣ AI内容幻觉检测器
**专业版 - 严谨报告风格**

检测AI生成内容中的：
- 🔴 幻觉和编造
- 🔴 伪科学表述
- 🔴 数据失实
- 🔴 引用造假

**使用场景**：内容审核、学术检验、媒体核查

**示例输出**：
```
⚠️ 高风险
发现3处幻觉：
1. 数据编造：引用的"研究"不存在
2. 伪科学表述：量子养生舱
3. 逻辑矛盾：前后数据不一致
```

---

### 2️⃣ AI营销骗局识别器
**趣味版 - 侦探断案风格**

检测营销文案中的：
- 🚨 虚假宣传话术
- 🚨 投资理财骗局
- 🚨 传销拉人头模式
- 🚨 价格欺诈套路

**五种侦探风格**：狄仁杰、包青天、柯南、福尔摩斯、秦风

**使用场景**：消费者防骗、网购决策、投资识别

**示例输出**：
```
【狄仁杰断案】
元芳，此事必有蹊跷...
经本官审理，此广告疑点共计 4 处。
最高风险等级：🚨 极高风险
本案判定：疑似诈骗

【维权建议】
1. 立即停止交易
2. 向公安机关报案（110）

元芳，你怎么看？
```

---

## 🆚 两个检测器的区别

| 对比项 | AI内容幻觉检测器 | AI营销骗局识别器 |
|--------|-----------------|-----------------|
| **风格** | 严谨报告 | 趣味断案 |
| **场景** | 学术/专业内容 | 消费/营销场景 |
| **输出** | 风险分级报告 | 维权建议+举报渠道 |
| **适用** | 内容创作者 | 消费者 |
