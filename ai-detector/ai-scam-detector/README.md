# AI营销骗局识别器

> 🦞 专注识别营销文案中的虚假信息和消费陷阱，保护你的钱包安全！

## 📋 简介

AI营销骗局识别器是一个趣味性与实用性兼备的技能，帮助消费者识别：
- 🏥 保健品虚假宣传
- 💰 投资理财骗局
- 🛒 网购商品真假
- 📱 传销话术识别
- 🎯 营销套路拆解


## 🚀 使用方法

### 方式一：命令行

```bash
python scripts/scam_detector.py
```

### 方式二：作为模块

```python
from scripts.scam_detector import ScamDetector

# 创建检测器（选择角色）
detector = ScamDetector("direnjie")  # 可选: direnjie, baoqingtian, conan, holmes, qinfeng

# 检测内容
content = "量子共振养生舱，7天根治糖尿病"
result = detector.detect(content)

# 格式化输出
output = detector.format_output(result)
print(output)
```

## ⚠️ 风险等级

| 等级 | 说明 | 建议 |
|------|------|------|
| 🚨 极高风险 | 疑似诈骗 | 立即停止交易，向公安机关报案 |
| ⚠️ 高风险 | 虚假宣传 | 谨慎购买，要求商家提供资质 |
| 🟡 中风险 | 夸大营销 | 货比三家，理性消费 |
| ✅ 低风险 | 基本可信 | 可正常购买，保留购物凭证 |

## 🆚 与幻觉检测器的区别

| 对比项 | AI内容幻觉检测器 | AI营销骗局识别器 |
