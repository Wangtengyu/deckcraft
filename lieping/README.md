# 🛋️ 人生躺平计算器

> 科学测算你的躺平所需本金，追踪财务进度，让FIRE生活不再遥远

![躺平计算器](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🎯 产品定位

**长期储蓄规划工具** - 不是一次性计算器，用户持续追踪进度直到躺平目标达成。

## ✨ 核心功能

### 📊 智能躺平测算
- **4%法则**：躺平本金 = 年支出 × 25
- **中国特色**：考虑2.5%通胀率
- **三种方案**：极简躺平 / 舒适躺平 / 富足躺平
- **复利曲线**：保守(3%) / 稳健(6%) / 进取(9%)

### 📈 财务数据管理
- **支出分类**：固定支出、日常支出、偶尔支出
- **收入计算**：主动收入 + 被动收入
- **城市数据**：50+城市的生活成本数据

### 🗺️ 阶段指导
- 0-2年：极端储蓄期
- 2-5年：复利加速期
- 5-10年：被动收入期
- 10年+：躺平准备期

### 🎨 趣味分享
- 个人财务体检报告风格
- 有趣对比："你在XX城市，躺平需要XXX万"
- 成就标签："储蓄率击败XX%年轻人"
- 进度图：存款增长曲线

### 🏆 里程碑系统
- 首个10万
- 储蓄率突破50%
- 被动收入过千
- 年度目标达成

## 📖 使用指南

### 首次使用
1. 打开首页，点击「开始测算」
2. 选择你所在的城市
3. 填写你的收入情况
4. 填写你的支出情况
5. 查看躺平报告

### 持续追踪
1. 每月添加存款记录
2. 查看进度更新
3. 解锁里程碑成就
4. 生成分享图激励好友

## 🔧 自定义配置

### 城市数据
在 `js/data.js` 中修改 `CITY_DATA` 对象，可以添加更多城市。

### 躺平方案
修改 `RETIREMENT_PLANS` 配置可以自定义不同方案。

### 里程碑
修改 `MILESTONES` 配置可以添加自定义里程碑。
